Dorkboard working with Ubuntu 8.04

By: PhilipOfOregon

2009-01-26 22:40:50

I needed to do a little extra work to get the Dorkboard working through Ubuntu 8.04.

I added a udev rule as described in a benito comment.

I also added a symbolic link: "ln -s ttyACM0 ttyUSB0" as suggested in a cult induction followup for rev3.

The symbolic link is necessary to get the Arduino IDE to find the USB serial port.
