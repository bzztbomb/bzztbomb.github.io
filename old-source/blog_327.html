Me vs. the Nordic Breakout Board 2009-03-29 22:39:28 skinny I've been attempting to help (or maybe hinder!) Don with a project he's working on. My task is to get a Dorkboard and eventually just the Benito talking to this Nordic nRF2401A breakout board from Sparkfun. After hacking on it a for a few days and today at the Open Lab and even more hacking at home I finally have the example program working! I'm attaching the source for the Arduino sketch for anyone else trying to do the same thing. Other things to note:
  • Link to the Code.
  • The Nordic board needs 3.3v. Don is running the Benito and Dorkboard at 3.3v (see here) to make things simple.
  • Pin config (Nordic board (Left to Right), Pin on Dorkboard, Arduino Pin Number for digitalWrite)
    • GND, 22, NA
    • DATA, 11, 5
    • CLK1, 6, 4
    • DR1, 5, 3
    • CS, 9, 6
    • CE, 13, 7
    • VCC, 7, NA
DataSheets:
  • Nordic RF2401A
  • ATMega 168v
Random lesson learned from this project so far: Think about timing, and remember that doing serial debugging statements will really putz up your timings! ;)
