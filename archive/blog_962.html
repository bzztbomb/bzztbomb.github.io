What?

I'm trying out a new drop in replacement editor for Drupal.  Trying to get my head around what this means and what to think about it, but I think it's mostly good.

Pros:

  • Easier to do simple entries
  • Looks nice while you're blogging/writing
  • Seems to work ok, mostly
  • Less markup to type
  • Can always fall back to plain editor (link is right below the editor itself)

Cons:

  • Needs some customization
  • Requires user to pick full html input format (although that might be a configuration directive)
  • Image picker is not working completely
  • Need to customize to allow more than one indent level.
